üíé Gems (Personalized Shortcuts)

Gems are personalized, pre-set prompts that allow you to save your most frequent and complex instructions. By creating a Gem, you establish a consistent format and tone for repeatable tasks, turning multi-step operations into a one-click shortcut.

  • Focus: Ensures a consistent format, style, and persona across repeated tasks.
  • Benefit: Maximizes efficiency by minimizing the need to re-type complex prompts.
  • Example: A "Create Lesson Plan" Gem that automatically follows your desired template.

üé® Canvas (Code & Educational Tool Generation)

Canvas allows you to generate complete code files, including interactive simulations, educational games, or simple data visualizations using HTML, CSS, and JavaScript. This makes it an ideal tool for creating customized learning resources that can be run instantly.

  • Application: Generate interactive code artifacts like Physics simulations, visual quizzes, or basic games for hands-on student practice.
  • Benefit: Provides immediate, runnable examples of complex concepts.
